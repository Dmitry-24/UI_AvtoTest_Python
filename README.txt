About Project:
Hi all! You can see automated user interface tests realization with  Selenium and Python for online bookstore website.
Build of test-cases using PageObject pattern, we have basic class(BasePage) and three child classes(ProductPage, LoginPage, MainPage).
Two files with test-cases: test_main_page.py and test_product_page.py. Test-cases include user actions on the web-page.
Tests run on Chrome or Mozilla Firefox and support multiple user languages else using Pytest framework.
You can input "pytest -v --tb=line --language=en -m need_review" to terminal for tests run or use other Pytest commands.(Some tests are marked).
